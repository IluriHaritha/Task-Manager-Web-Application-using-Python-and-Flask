# 📝 Task Manager Web Application

## 📌 Project Overview
This is a simple Task Manager web application built using Python and Flask.  
It allows users to add, view, and delete daily tasks with persistent storage using SQLite database.

---

## 🚀 Live Demo
👉 [https://task-manager-web-application-using-xrwz.onrender.com/]

---

## 🛠️ Technologies Used
- Python
- Flask (Web Framework)
- SQLite (Database)
- HTML
- CSS

---

## 🎯 Features
- Add new tasks
- View all tasks
- Delete tasks
- Prevent duplicate tasks
- Flash messages for user feedback
- Persistent data storage using database
