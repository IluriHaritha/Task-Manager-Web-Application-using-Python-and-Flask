# Task Manager Web Application

## 📌 Description
A simple web application to manage daily tasks.

## 🚀 Features
- Add tasks
- View tasks
- Delete tasks
- Prevent duplicate tasks
- Flash messages for user feedback

## 🛠 Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS

## 📂 Project Structure
task_manager/
│
├── app.py
├── templates/
│     └── index.html
├── tasks.db

## ▶️ How to Run
1. Install Flask
2. Run command: python app.py
3. Open browser: http://127.0.0.1:5000/

## 🎯 Purpose
This project is built for learning backend development and web application fundamentals.
